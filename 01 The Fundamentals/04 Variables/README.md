# Variables

This lesson covers how to define and use variables in PHP.

## Key Concepts
- **Variable Syntax**: Variables in PHP start with a `$` sign (e.g., `$greeting`).
- **Assignment**: Use the `=` operator to assign values to variables.
- **Concatenation**: The dot `.` operator is used to join strings together.
- **String Interpolation**: Variables can be expanded inside double quotes, but not single quotes.

## Code Example
```php
$gathering = "Hello";
$name = "Everybody";
echo $gathering . " " . $name . "!"; // Outputs: Hello Everybody!
```

# Tools of the Trade

This section outlines the essential tools required for PHP development found in `tool.txt`.

## Editors
- **VS Code**: A popular, extensible code editor.
- **Antigravity**: An AI-powered coding assistant and editor.

## Local Development Environments
These tools bundle PHP, MySQL, and other necessary components:
- **XAMPP**: Cross-Platform (X), Apache (A), MySQL (M), PHP (P) and Perl (P).
- **Laragon**: A modern, isolated, and portable development environment for Windows.

## Command Line
- **Windows Command Prompt**: The default command-line interpreter.
- **Wamp**: Windows web development environment (also includes command line tools).

## Database Management
- **phpMyAdmin**: A web interface for managing MySQL databases.
- **TablePlus**: A modern, native tool for managing multiple databases.
